# 📄 AI Resume Analyzer

## 🚀 Overview
AI Resume Analyzer is a Streamlit-based web application that automates resume screening by extracting information from PDF resumes. It identifies key technical skills, calculates an ATS-style score, and provides personalized feedback such as missing skills and interview questions based on the candidate profile.

This project simulates a simplified **Applicant Tracking System (ATS)** used in real-world recruitment workflows.

---

## 🎯 Problem Statement
Manual resume screening is time-consuming, inconsistent, and not scalable. Recruiters need an automated system that can quickly evaluate candidates based on skills and job relevance.

---

## 💡 Solution
This system automates resume evaluation by:
- Extracting text from PDF resumes
- Detecting technical skills using keyword matching
- Calculating an ATS-style compatibility score
- Identifying missing skills for improvement
- Generating role-based interview questions
- Providing a structured resume summary

---

## ✨ Features
- 📄 PDF resume upload and parsing
- 🧠 Automated skill extraction
- 📊 ATS-style scoring system
- ⚠️ Missing skill identification
- ❓ Interview question generation
- 📝 Resume summary generation
- 🎯 Job role selection for role-based evaluation
