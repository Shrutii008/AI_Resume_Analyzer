# AI Resume Analyzer

A Streamlit-based AI Resume Analyzer that extracts resume text, detects skills, calculates an ATS score, identifies missing skills, generates interview questions, and allows users to download a report.

## Features
- Resume PDF Upload
- Skill Detection
- ATS Score Calculation
- Missing Skills Analysis
- Interview Question Generation
- Resume Summary
- Downloadable Report

## Installation

pip install -r requirements.txt

## Run the Application

streamlit run app.py
